# Esp_Hand
A bionic arm powered by an ESP32 and EMG sensors.

I’ve always been fascinated by EMG sensors and how they can capture muscle activity and convert it into meaningful control signals. I really wanted to build something practical using them. Around the same time, I came across the ESP-NOW protocol on the ESP32, which enables fast and efficient wireless communication without needing Wi-Fi. That made the idea even more interesting.

So I decided to combine both of these into a single project.

Esp_Hand is an attempt to build a simple bionic arm that can respond to muscle signals. The EMG sensor reads electrical activity generated by muscle contractions and outputs an analog signal. This signal is then processed by the ESP32 to detect movements or gestures, which are used to control the arm.

ESP-NOW is a lightweight, low-latency wireless communication protocol developed by Espressif. It allows ESP32 devices to communicate directly with each other without a router, making it ideal for real-time applications like this where quick response is important. In this project, it helps in transmitting control data wirelessly between modules.

This project helped me explore:

- EMG signal acquisition and basic processing
- Real-time control using microcontrollers
- Wireless communication using ESP-NOW
- Integrating hardware and software into a working system

The goal is not just to build a prototype, but to understand how human-machine interaction works at a basic level and how it can be improved in the future.
